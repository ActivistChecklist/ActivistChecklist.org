---
title: Use a unique identity for every online account you make (name, email, username, phone number)
slug: alias
---

Instead of signing up with your real name/email/phone/etc., you can start using **random, unique identities for each account**. 

This makes it harder for attackers (and tech companies) to link your accounts across sites and create a profile from all those bits of information.

<HowTo title="How to use a unique identity for each account">

Make sure each of these is unique for each site, whenever feasible: 

- **Display name:** [Use this random identity generator](https://strongphrase.net/#/identity).
- **Username:** [Use this identity generator](https://strongphrase.net/#/identity) or this specific [username generator](https://strongphrase.net/#/username).
- **Profile photo:** Leave it blank or use [this identity generator](https://strongphrase.net/#/identity). If you use the same photo on all sites, an attacker can do a "reverse image search" to find all your accounts.
- **Email address** - Use an email alias service like: 
- [Qwacky to use @duck.com email alias](https://github.com/Lanshuns/Qwacky) (free, Chrome/Firefox extension)
- [Proton Mail](https://proton.me/support/pass-email-alias) (free/paid)
- [Addy.io](https://Addy.io) (free/paid)
- **Phone number:** Try to just not enter a phone number when prompted. If you need to enter a phone number just for verification when you sign up, try renting a phone number from [SMSPool](https://www.smspool.net/) ($0.50 per registration). If you need a long-term phone number, consider [MySudo](https://anonyome.com/individuals/mysudo/).

</HowTo>
