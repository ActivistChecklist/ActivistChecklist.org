---
title: Install a secure browser
---

- Install [Onion Browser](https://apps.apple.com/us/app/onion-browser/id519296448) (iPhone) or [Tor Browser](https://www.torproject.org/download/#android) (Android) for maximum privacy.
- Or at least install [Brave Browser](https://brave.com/en-in/download/#brave-mobile) to use instead of the default browser.
