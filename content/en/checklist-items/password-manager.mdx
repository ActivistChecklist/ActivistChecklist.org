Our main recommendations are:

* **1Password**: Very user friendly. Slightly more secure. Costs $3/month
* **Bitwarden**: Free. Still quite secure.

<HowTo title="How to set up 1Password">
  1. **Download:** [Download and install 1Password](https://support.1password.com/get-the-apps/) ($3/month)
  2. **Master password:** Create a strong, random "master password" using a [passphrase generator](https://strongphrase.net/). It should be memorable, but not a password you use anywhere else. Write your master password down on paper rather than storing it digitally. Set a reminder to destroy the paper in a few weeks once you have it memorized.
  3. **Import:** [Import your existing passwords](https://support.1password.com/import/) from your computer or browser
  4. **Apps:** [Install the browser extension](https://1password.com/downloads/browser-extension/) and mobile app ([iPhone](https://1password.com/downloads/ios), [Android](https://1password.com/downloads/android)) to help you save and auto-fill passwords
  5. **Change passwords:** If you had been re-using similar passwords, update your most important ones using the random password generator built-in to 1Password.

  See 1Password's [getting started guide](https://support.1password.com/explore/get-started/) for a video of these steps.
</HowTo>

Bonus: Here’s a good introduction on how to [get the most out of 1Password](https://www.nytimes.com/wirecutter/guides/how-to-use-1password/).

**Alternative options:**

* [Proton Pass](https://proton.me/pass): has a free option
* [KeePassXC:](https://keepassxc.org/) Open-source and allows you to store passwords only on your machine instead of the cloud, but the user interface is very clunky.
